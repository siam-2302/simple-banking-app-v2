About SimpleBankApp

Our Mission

SimpleBankApp provides a secure, intuitive, and reliable digital banking experience. Our mission is to make financial services accessible while maintaining high standards of security and user experience.

Who We Are

We are a team of student developers and fintech enthusiasts committed to developing modern web-based banking solutions. This project demonstrates secure app development practices for educational purposes.

Key Features

  • Secure user authentication
  • Role-based access control
  • Real-time money transfers
  • Transaction history tracking
  • Admin and manager dashboards
  • Philippine location data via PSGC API

Security Measures

  • Hashed passwords with bcrypt
  • CSRF protection on all forms
  • Rate limiting to prevent abuse
  • Session security (HttpOnly, SameSite)
  • Two-step transaction confirmation
  • Role-based admin approval workflows

Technology Stack

Backend

Python with Flask

Database

MySQL with SQLAlchemy ORM

Frontend

HTML, CSS, Bootstrap 5

Security

Flask-WTF, Flask-Bcrypt, Flask-Limiter
